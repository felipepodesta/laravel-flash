## Installation

To install in Laravel (current):

```sh
composer require felipepodesta/laravel-flash
```

Add the service provider to your `config/app.php` file:

```php
<?php
/*
    'providers' => [
        [...]
*/
        FelipePodesta\Flash\ServiceProvider::class,
/*
        [...]
    ],
*/
```
